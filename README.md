# Professional Portfolio - Mateo Castro

Welcome to my personal portfolio project. This repository contains a modern, responsive, and high-performance portfolio website built with **React** and **Vite**. It is designed to showcase my skills, projects, and professional background as a Junior Web Developer in training.

![Portfolio Preview](public/favicon.svg)

## 🚀 Overview

The portfolio serves as my digital business card, featuring a clean design and smooth navigation to provide a seamless user experience. It highlights my journey as a 2nd-year Web Application Development student (DAW) and my readiness for professional internships.

## ✨ Features

-   **Responsive Design**: Optimized for desktops, tablets, and mobile devices.
-   **Dynamic Navigation**: Smooth scrolling and React-based routing for a single-page feel.
-   **Interactive Projects Showcase**: A filtered grid displaying multiple projects with direct links to GitHub.
-   **Skill Visualization**: A visual representation of technical proficiencies and current learning path.
-   **Direct Contact & CV Download**: A modern contact section with a quick CV download feature integrated with assets.
-   **Custom Branding**: Personalized custom favicon and typography.

## 🛠️ Tech Stack

### Frontend
-   **React 19**: Modern UI library for building interactive components.
-   **Vite**: Next-generation frontend tooling for fast builds and HMR.
-   **React Router 7**: Robust routing for managing application states.
-   **CSS3**: Custom vanilla CSS for high-performance, polished aesthetics (no heavy frameworks).

### Tools & Others
-   **Git/GitHub**: Version control and hosting.
-   **Figma**: UI/UX design and prototyping.

## 💻 Getting Started

### Prerequisites
-   [Node.js](https://nodejs.org/) (v18 or higher recommended)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/MateoCastro47/Porfolio_Online.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Porfolio_Online
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Development

Run the development server:
```bash
npm run dev
```
Open `http://localhost:5173` in your browser.

### Build

Create a production-ready bundle:
```bash
npm run build
```
